Welcome to a fan created, unofficial Warframe data viewer. This webapp works by parsing the official API and displaying the data it contains. Think of this as kind of like an out of game Codex.

If you want to see a particular category tackled next, let me know and I will see what I can do to implement it sooner. Please note that this is an open source, one man project right now so things might be slow moving. Contributions are welcome however, more information can be found here.

Categories available in the API:

  • All (will never be implemented, the page would be massive and unusable)
  • Arcanes
  • Archwing
  • Fish
  • Gear
  • Glyphs
  • Melee
  • Misc
  • Mods
  • Pets
  • Primary
  • Quests
  • Relics
  • Resources
  • Secondary
  • Sentinels
  • Sigils
  • Skins
  • Warframes
